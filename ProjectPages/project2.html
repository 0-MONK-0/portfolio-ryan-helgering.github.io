Five Doors

This assignment was to create a presentation of a number of personal projects that were selected to illustrate core values of pride, fun, self-esteem, growth and individuality as a reflection on a year of self-development. This project was a rewarding challenge due to the ambitious goal I set myself, with the attention to detail I wanted to build into 5 presentations in game format. I hit many technical challenges but was able to solve these in the end. I also created a trailer to give this project an extra professional presentation.
