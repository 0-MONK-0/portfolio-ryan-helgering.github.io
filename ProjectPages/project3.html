Five Doors

This project was an early development in Unity focused on creating a fun and interactive game with the freedom to choose the format and style. I chose to branch out into making a first-person parkour game because I wanted to try something new and challenging with more focus on environment and object dynamics, physics and rendering. This project was motivating because it stretched my abilities in Unity whilst enabling me to experiment further on the interplay between game design, 3-D development and visual rendering and effects.
